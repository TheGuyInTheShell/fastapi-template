# Registro y control de ingresos y egresos de los empleados de la vit a traves de reconocimiento facial usando la libreria DeepFace

## Instalación


1. Clonar el repositorio:
git clone https://github.com/TheGuyInTheShell/reconocimiento-facial-backend.git


2. Crear un entorno virtual e instalar las dependencias:
python3 -m venv env source env/bin/activate pip install -r requirements.txt

3. Borrar el .example de .env.example


## Ejecución

1. Ejecutar el servidor de desarrollo:
uvicorn main:app --reload
